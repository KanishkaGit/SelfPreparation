Abstract Classes and Abstract Methods

Abstract Methods

Abstract Methods are methods without a definition. So, a method without any method body is known as an Abstract Method. What it contains is only declaration of the method. 


                    //not an Abstract Method
                    public void Add(int x, int y){
                        //Method Body
                    }

                    //Abstract Method
                    public abstract void Add(int x, int y);
                    //Abstract Methods are to be defined by using abstract keywords
